### Template Context: SPOT_THE_MISTAKE

The resulting visualization will be implemented as:
- Identify and highlight errors in code, text, or diagrams.
- Content is displayed with clickable regions.
- Mistakes are highlighted when identified.

## Template Component Inventory

SPOT_THE_MISTAKE template provides the following components that can be customized for any question:

### Available Components:

1. **Content Components**:
   - **Content Display**: Code, text, or diagram display area (adapts to question context)
   - **Clickable Regions**: Regions that can be clicked to identify mistakes
   - **Content Styling**: Colors, fonts, borders for content display

2. **Interactive Elements**:
   - **Region Click**: Click to identify mistake
   - **Mistake Highlighting**: Visual highlighting of identified mistakes
   - **Submit Button**: Button to submit identified mistakes
   - **Reset Button**: Button to reset selections

3. **Feedback Elements**:
   - **Correct Identification Feedback**: Visual feedback when mistake is correctly identified
   - **Incorrect Identification Feedback**: Visual feedback when non-mistake is clicked
   - **Success Message**: Appears when all mistakes are identified
   - **Mistake Highlighting**: Visual highlighting of mistakes

4. **Layout Structure**:
   - **Question Container**: Top area for instructions/question text
   - **Content Pane**: Main area for content display
   - **Controls Pane**: Bottom area for submit/reset buttons

5. **Animation Components**:
   - **Region Click Animation**: Visual feedback when region is clicked
   - **Mistake Highlight Animation**: Visual highlighting of mistakes
   - **Correct Identification Animation**: Success animation when mistake is correctly identified
   - **Incorrect Identification Animation**: Error animation when non-mistake is clicked
