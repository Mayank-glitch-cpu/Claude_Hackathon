### Template Context: PROBABILITY_LAB

The resulting visualization will be implemented as:
- Interactive probability experiments and simulations.
- Experiments are runnable simulations.
- Results are visualized in charts or graphs.

## Template Component Inventory

PROBABILITY_LAB template provides the following components that can be customized for any question:

### Available Components:

1. **Experiment Components**:
   - **Experiment Display**: Visual representation of experiment (adapts to question - coin flip, dice roll, spinner, etc.)
   - **Experiment Parameters**: Parameters that can be adjusted (number of trials, probability, etc.)
   - **Experiment Styling**: Colors, borders, shadows for experiment display

2. **Parameter Controls**:
   - **Parameter Sliders**: Sliders to adjust experiment parameters
   - **Parameter Inputs**: Input fields for parameter values
   - **Run Button**: Button to run experiment
   - **Reset Button**: Button to reset experiment

3. **Result Components**:
   - **Result Charts**: Charts or graphs showing experiment results
   - **Result Statistics**: Statistical measures (mean, variance, etc.)
   - **Result Display**: Text or visual display of results
   - **Result Styling**: Colors, fonts, borders for result display

4. **Interactive Elements**:
   - **Parameter Adjustment**: Adjust experiment parameters
   - **Experiment Run**: Run experiment simulation
   - **Result Visualization**: Visualize experiment results
   - **Submit Button**: Button to submit answer

5. **Feedback Elements**:
   - **Correct Answer Feedback**: Visual feedback when answer is correct
   - **Incorrect Answer Feedback**: Visual feedback when answer is incorrect
   - **Success Message**: Appears when answer is correct
   - **Result Highlighting**: Visual highlighting of results

6. **Layout Structure**:
   - **Question Container**: Top area for instructions/question text
   - **Experiment Pane**: Main area for experiment display
   - **Parameters Pane**: Area for parameter controls
   - **Results Pane**: Area for result visualization
   - **Controls Pane**: Bottom area for run/submit/reset buttons

7. **Animation Components**:
   - **Experiment Run Animation**: Visual feedback when experiment runs
   - **Result Update Animation**: Visual feedback when results update
   - **Statistical Visualization Animation**: Visual progression of statistical visualization
   - **Answer Validation Animation**: Success/error animation on validation
