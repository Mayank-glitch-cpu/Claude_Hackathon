### Template Context: SEQUENCE_BUILDER

The resulting visualization will be implemented as:
- Drag-and-drop interface to arrange steps in correct order.
- Steps are represented as cards that can be reordered.
- Correct ordering is validated on submission.

## Template Component Inventory

SEQUENCE_BUILDER template provides the following components that can be customized for any question:

### Available Components:

1. **Step Components**:
   - **Step Cards**: Draggable cards representing steps in a sequence
   - **Step Content**: Text or images on step cards
   - **Step Ordering**: Correct order index for validation

2. **Interactive Elements**:
   - **Drag-and-Drop**: Drag steps to reorder them
   - **Drop Zones**: Areas where steps can be placed
   - **Submit Button**: Button to validate sequence
   - **Reset Button**: Button to reset order

3. **Feedback Elements**:
   - **Correct Placement Feedback**: Visual feedback when step is in correct position
   - **Incorrect Placement Feedback**: Visual feedback when step is in wrong position
   - **Success Message**: Appears when sequence is correct
   - **Step Highlighting**: Visual highlighting of steps

4. **Layout Structure**:
   - **Question Container**: Top area for instructions/question text
   - **Steps Pane**: Main area for step cards (unordered initially)
   - **Sequence Pane**: Area for ordered steps (drop zones)
   - **Controls Pane**: Bottom area for submit/reset buttons

5. **Animation Components**:
   - **Step Drag Animation**: Smooth dragging animation
   - **Correct Placement Animation**: Success effect when step is correctly placed
   - **Incorrect Placement Animation**: Error effect when step is incorrectly placed
   - **Sequence Complete Animation**: Success animation when all steps are correctly ordered
