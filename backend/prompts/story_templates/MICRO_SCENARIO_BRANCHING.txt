### Template Context: MICRO_SCENARIO_BRANCHING

The resulting visualization will be implemented as:
- Interactive branching scenarios with decision points.
- Scenarios present choices that lead to different branches.
- Path taken is tracked and evaluated.

## Template Component Inventory

MICRO_SCENARIO_BRANCHING template provides the following components that can be customized for any question:

### Available Components:

1. **Scenario Components**:
   - **Scenario Display**: Text or visual representation of current scenario
   - **Scenario Images**: Images representing scenarios (optional)
   - **Scenario Styling**: Colors, fonts, borders for scenario display

2. **Choice Components**:
   - **Choice Buttons**: Buttons representing available choices
   - **Choice Text**: Text labels on choice buttons
   - **Choice Styling**: Colors, borders, shadows for choice buttons

3. **Branch Components**:
   - **Branch Paths**: Visual representation of branching paths
   - **Path Tracking**: Visual tracking of path taken
   - **Branch Styling**: Colors, lines, arrows for branches

4. **Interactive Elements**:
   - **Choice Selection**: Click to select a choice
   - **Scenario Navigation**: Navigate through scenarios
   - **Path Visualization**: Visualize branching paths
   - **Reset Button**: Button to restart scenario

5. **Feedback Elements**:
   - **Correct Path Feedback**: Visual feedback when correct path is taken
   - **Incorrect Path Feedback**: Visual feedback when incorrect path is taken
   - **Success Message**: Appears when correct path is completed
   - **Choice Highlighting**: Visual highlighting of selected choices

6. **Layout Structure**:
   - **Question Container**: Top area for instructions/question text
   - **Scenario Pane**: Main area for scenario display
   - **Choices Pane**: Area for choice buttons
   - **Path Pane**: Area for path visualization (if needed)
   - **Controls Pane**: Bottom area for navigation/reset buttons

7. **Animation Components**:
   - **Scenario Transition Animation**: Visual transition between scenarios
   - **Choice Selection Animation**: Visual feedback when choice is selected
   - **Path Visualization Animation**: Visual progression along path
   - **Path Completion Animation**: Success animation when path is completed
