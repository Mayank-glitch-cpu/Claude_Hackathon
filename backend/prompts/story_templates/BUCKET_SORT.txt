### Template Context: BUCKET_SORT

The resulting visualization will be implemented as:
- Categorization by dragging items into appropriate buckets.
- Multiple buckets represent different categories.
- Items belong to specific categories.

## Template Component Inventory

BUCKET_SORT template provides the following components that can be customized for any question:

### Available Components:

1. **Bucket Components**:
   - **Buckets**: Containers representing different categories
   - **Bucket Labels**: Category names or labels on buckets
   - **Bucket Styling**: Colors, borders, shadows for each category

2. **Item Components**:
   - **Items**: Draggable items to be categorized
   - **Item Content**: Text or images on items
   - **Item Styling**: Colors, borders, shadows for items

3. **Interactive Elements**:
   - **Item Drag**: Drag items to buckets
   - **Drop Zones**: Bucket areas where items can be dropped
   - **Submit Button**: Button to validate categorization
   - **Reset Button**: Button to reset items

4. **Feedback Elements**:
   - **Correct Placement Feedback**: Visual feedback when item is in correct bucket
   - **Incorrect Placement Feedback**: Visual feedback when item is in wrong bucket
   - **Success Message**: Appears when all items are correctly categorized
   - **Bucket Highlighting**: Visual highlighting of buckets

5. **Layout Structure**:
   - **Question Container**: Top area for instructions/question text
   - **Items Pane**: Area for draggable items (unordered initially)
   - **Buckets Pane**: Main area for bucket display
   - **Controls Pane**: Bottom area for submit/reset buttons

6. **Animation Components**:
   - **Item Drag Animation**: Smooth dragging animation
   - **Bucket Drop Animation**: Visual feedback when item is dropped
   - **Correct Placement Animation**: Success effect when item is correctly placed
   - **Incorrect Placement Animation**: Error effect when item is incorrectly placed
